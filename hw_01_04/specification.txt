Спецификация модуля A1

Имя модуля: reading
Функция модуля: считывание строк-чисел исходного файла
Список параметров: файловая переменная исходного файла; число из очередной строки
Входные данные: файловая переменные
Выходные данные: строка без символа переноса строки
Внешние эффекты: перемещение указателя в файле на следующую строку



Спецификация модуля A2
Имя модуля: system
Функция модуля: проверка того, что число является двоичным
Список параметров: строка с числом
Входные данные: строка с числом
Выходные данные: нет
Внешние эффекты: возвращение истинности или ложности



Спецификация модуля A3

Имя модуля: convert
Функция модуля: перевод двоичного числа в шестнадцатеричное
Список параметров: строка с числом в двоичной системе счисления; строка с числом в шестнадцатеричной системе счисления
Входные данные: строка с числом в двоичной системе счисления
Выходные данные: строка с числом в шестнадцатеричной системе счисления
Внешние эффекты: нет



Спецификация модуля A4

Имя модуля: convert
Функция модуля: запись полученного результата в новый файл
Список параметров: строка с числом в шестнадцатеричной системе счисления; текстовый файл со строками в шестнадцатеричной системе счисления
Входные данные: строка с числом в шестнадцатеричной системе счисления
Выходные данные: текстовый файл со строками в шестнадцатеричной системе счисления
Внешние эффекты: перемещение указателя в файле на следующую строку

Входные данные: строка с целым двоичным числом

Внешние эффекты: нет



Спецификация модуля A32

Имя модуля: adding

Список параметров: строка с целым двоичным числом; строка с числом, готовым к переводу
Входные данные: строка с целым двоичным числом
Выходные данные: строка с числом, готовым к переводу
Внешние эффекты: нет



Спецификация модуля A33

Имя модуля: convert
Функция модуля: перевод числа по тетрадам в шестнадцатеричное
Список параметров: строка с числом, готовым к переводу; строка с числом в шестнадцатеричной системе счисления
Входные данные: число, готовое к переводу
Выходные данные: строка с числом в шестнадцатеричной системе счисления
